# Score Comparison Module

This module will allow users to compare their performance against national or group averages.

## Planned Features:
- Compare scores with national averages
- Regional performance comparison
- Percentile ranking
- Benchmark visualization
- Demographic comparisons

## Future Implementation:
- `ComparisonChart.tsx` - Visual comparison component
- `NationalAverage.tsx` - Display national average data
- `useComparison.ts` - Hook for comparison data
- `comparisonService.ts` - API service for comparison metrics
