# School Recommendation Module

This module will provide personalized school recommendations based on exam performance.

## Planned Features:
- School matching algorithm
- Admission probability calculator
- Requirements checker
- Application tracking
- School comparison tool

## Future Implementation:
- `SchoolRecommendations.tsx` - Main recommendations component
- `SchoolCard.tsx` - Individual school display
- `AdmissionCalculator.tsx` - Probability calculator
- `useRecommendations.ts` - Hook for recommendation logic
- `schoolService.ts` - API service for school data
